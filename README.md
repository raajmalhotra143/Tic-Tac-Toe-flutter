**Neon Grid** is a premium, fully-animated 3D Tic Tac Toe mobile game built with Flutter. Experience the classic game with a modern neon twist, featuring glassmorphism, dynamic glowing effects, and fluid 60 FPS animations.

## ✨ Features

* **Futuristic 3D UI:** Glassmorphism panels, deep shadows, and immersive neon glowing gradients.
* **Cinematic Animations:** Smooth page transitions, floating particles, and tactile button press effects powered by `flutter_animate`.
* **Custom AI & Campaign:** Play offline against an advanced Robot AI across multiple difficulty levels, or unlock stages in Campaign Mode.
* **Customizable Players:** Ditch the standard 'X' and 'O'. Choose from a variety of fun emojis (😀, 😎, 👻, 🔥, etc.) to represent your moves.
* **Animated Board:** 3D elevated tiles that physically depress when tapped and intensely glow when a winning line is achieved.
* **Premium Splash Experience:** Custom native splash screen seamlessly transitioning into a futuristic loading sequence.

## 🛠️ Tech Stack

* **Framework:** [Flutter](https://flutter.dev/)
* **State Management:** [Riverpod](https://riverpod.dev/)
* **Routing:** [go_router](https://pub.dev/packages/go_router)
* **Animations:** [flutter_animate](https://pub.dev/packages/flutter_animate)
* **Local Storage:** [Isar Database](https://isar.dev/)
* **Backend Ready:** Supabase integrated for multiplayer & auth.

## 🚀 Getting Started

### Prerequisites
* Flutter SDK (v3.19.0 or higher)
* Dart SDK

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/raajmalhotra143/Tic-Tac-Toe-flutter.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Tic-Tac-Toe-flutter
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run code generation for Riverpod and Isar:
   ```bash
   dart run build_runner build --delete-conflicting-outputs
   ```
5. Run the app:
   ```bash
   flutter run
   ```
